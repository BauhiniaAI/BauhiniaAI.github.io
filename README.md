##### commit格式：

* [name1, name2] {type}-{StoryNo.}： commit description

* type前有空格，冒号后有空格

##### type类型：

* feat : 增加新功能

* fix: 修改问题
* refactor: 重构

##### css：

* 两格缩进
* { 前空格
* style外嵌式
